About Us

Bienvenidos

Nosotros somos el Equipo 8. Este es nuestro Proyecto Digital correspondiente a la fase 2 de BEDU. Aquí plasmamos los aprendizajes adquiridos durante el trayecto de esta fase al aplicar las hablidades aprendidas.

Este proyecto final es un buscador de recetas que cuenta principalmente con las siguientes especificaciones:

  1. Muestra en la User Interface (UI) un input para ingresar texto y un botón para generar la búsqueda de recetas.

  2. Utiliza la APIAPI Free Meal.

  3. Muestra los resultados usando las imágenes de las recetas.

  4. Al seleccionar una imagen se despliegan los detalles de la receta (ingredientes e instrucciones)

  5. Se Muestra en la UI un botón secundario para desplegar una receta aleatoria.
